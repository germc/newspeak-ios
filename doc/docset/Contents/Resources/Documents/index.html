    newspeak for ios

    unit tests are done with kiwi, integration tests are done with kif.

    for local testing, use the virtual machine from newspeak server.

    install instructions

    you need the osx package manager homebrew to install dependencies.

    install homebrew first:

    ruby -e "$(curl -fsSL https://raw.github.com/mxcl/homebrew/go)"

    then, proceed with the additional dependencies

    brew install mogenerator uncrustify appledoc
gem install cocoapods

    appledoc

    api documentation can be generated with appledoc.

    generated documentation is at doc/html/index.html.

    run appledoc with the command: ./appledoc.sh

    uncrustify

    uncrustify is used for consistent source code formatting.

    you can edit the uncrustify config file with uncrustify x. the uncrustify config file is based on this config file.

    run uncrustify with the command: ./uncrustify.sh
